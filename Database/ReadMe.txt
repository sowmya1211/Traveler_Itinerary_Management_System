This folder contains all the database related information that was used in the application
MySQL was used to store, retrieve and modify the tables

Run the 4 sql files in this folder to create and save all the tables in your local MySql database

1. CreateTables - used to create the required tables
2. InsertRecords_Accommodation - to store accommodation details required in this application
3. InsertRecords_Transit - to store transit details required in this application
4. InsertRecords_UsersPlaces - to store tourist places details required in this application
